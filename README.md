# GitHub API Client

Showcase for Kotlin, MVVM, Retroft, GreenDAO, RxJava, Dagger.

Shows repository search results. Offline first with local cache in SQLite database.

## Technology stack
- Kotlin
- Androidx
- MVVM
- [Retrofit]
- [GreenDAO]
- [RxJava]
- [Dagger]
